# AI Evaluator Assessment Rubric

A structured scoring rubric designed to assess the competencies of AI response evaluators. This framework measures the ability to identify biases, apply evaluation guidelines consistently, deliver actionable feedback, calibrate scores, and reflect metacognitively on the evaluation process.

---

## Purpose

This rubric was developed to support the selection and training of AI evaluators working on large language model (LLM) response assessment tasks. It provides a standardized, observable framework for measuring evaluator quality across five key competencies.

It can be used for:
- Candidate screening in hiring processes for AI evaluation roles
- Evaluator training and calibration sessions
- Self-assessment and portfolio documentation

---

## Rubric Structure

The rubric covers **5 evaluation criteria**, each scored on a **1–4 scale**:

| Score | Label | Description |
|-------|-------------|--------------------------------------|
| 4 | Exemplary | Complete, with concrete examples |
| 3 | Advanced | Mostly complete, one element missing |
| 2 | Beginner | Partial, key elements missing |
| 1 | Inadequate | Vague, absent, or off-topic |

---

## Criteria Overview

### 1. Bias Identification and Management
The candidate names at least 2 biases, describes how each could have altered their judgment, and identifies at least one countermeasure adopted.

### 2. Guideline Adherence
The candidate cites the most relevant criterion for the response evaluated, justifies the score based on received instructions rather than personal opinion.

### 3. Written Feedback Quality
The candidate uses clear language, references specific parts of the evaluated text with timestamps, indicating for each the issue found and a possible solution.

### 4. Score Calibration
The candidate provides consistent scores for similar responses and, when scores differ, justifies the difference with specific references to the text.

### 5. Metacognition
The candidate confirms having followed the guidelines, acknowledges that their personal experience may have influenced the evaluation, and identifies the specific bias that may have played a role.
